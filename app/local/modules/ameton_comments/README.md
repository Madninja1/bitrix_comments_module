# Ameton Comments (Bitrix module)

Система комментариев для страницы новости `/news/{id}/` с поддержкой:
- неограниченной вложенности (в рамках ТЗ — до 10 уровней при генерации)
- постраничной навигации: 10 комментариев 1-го уровня
- подгрузки следующего уровня через AJAX
- подсчёта количества ответов на всех подуровнях
- генерации тестовых данных (agent/CLI)

## Требования
- Bitrix (любая редакция)
- MySQL 8.0

## Установка
1. Скопировать модуль в `/bitrix/modules/ameton_comments/` или в `/local/modules/ameton_comments/`
2. Установить модуль через админку Bitrix
3. Открыть:
    - `/news/1/`
    - `/news/2/`
    - `/news/3/`
    - `/news/4/`
    - `/news/5/`

## Генерация данных
### Через админку
В админке модуля запустить сидирование (по шагам, через агент).

### Через CLI
Пример:
`php /bitrix/modules/ameton_comments/tools/seed.php --news=5`

## Архитектура (черновик)
- MySQL таблицы: comments + closure table + seed runs
- D7 ORM DataManager
- Компонент: рендер страницы + AJAX подгрузка уровней
